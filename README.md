Vue CLI v4.5.15 - Starter Vue Project
-----

Demo is [here](https://webdevelopua.github.io/vue-cli-v4/)

``` 
npm i @vue/cli -g

vue create .
```

* Please pick a preset: Default (Vue 3) ([Vue 3] babel, eslint)
    - Use Yarn
    - Use NPM

``` 
npm run serve
```

App running at:
- Local:   http://localhost:8080/
- Network: http://192.168.1.104:8080/
